Projects

Computer Programming IoT Capstone Project

As part of my Computer Programming IoT final capstone project, I contributed to the design and development of a 3D-printed, remotely operated vehicle built to transport plastic balls across a room. The project introduced a unique challenge: the operator could not directly see the vehicle and instead relied entirely on a live camera feed from a separate room.

My primary role focused on the software and systems integration side of the project. I worked on implementing the live camera feed, developing the communication system between the Raspberry Pi and the client interface, and linking client-side input controls to the vehicle's motor systems. This included integrating control logic for the stepper motor (bucket mechanism), servo motor (steering), and DC motor (throttle). I also helped build the client interface used to both control the vehicle and monitor its live video feed.

This project strengthened my experience in software development, embedded systems, client-server communication, and hardware-software integration, while also demonstrating my ability to contribute to complex, hands-on technical solutions.

Butterfly Room Environment Control System (API Capstone Project)

As part of an API capstone project, I contributed to the design and development of an environment control system for a butterfly room, created to monitor and maintain safe living conditions. The system tracks temperature and humidity using a DHT11 sensor, allowing users to define target ranges for each value. When the room's conditions move outside those ranges, the system automatically generates alerts and triggers connected controls to restore the environment.

My primary contributions focused on the REST API and alerting system. I helped develop API endpoints that allow users to retrieve all sensor data or query individual sensors, with support for pagination and configurable page sizes for efficient data access. I also researched and implemented the Twilio SMS API, enabling the system to send real-time text alerts to a designated phone number whenever temperature or humidity levels deviate from the preset targets.

In addition, I assisted with the MQTT communication layer, which handles messaging between sensors and actuators such as heaters and dehumidifiers. This included supporting JSON-based device commands and helping ensure the system could safely respond to changes in the environment. The project strengthened my experience in RESTful API design, third-party API integration, IoT communication protocols, and fail-safe system development.

CampusCafeteriaDB – Software Engineering Database Project

As part of a software engineering course focused on modern development practices and the full software development lifecycle, I designed and developed CampusCafeteriaDB, a structured relational database built to support a modern campus cafeteria management system. The project was created as a backend foundation for applications such as web platforms, mobile apps, point-of-sale systems, and administrative dashboards, with support for user management, menu tracking, orders, payments, promotions, customer feedback, and support services.

The project emphasized both technical implementation and software engineering principles. I designed a normalized relational schema with clearly defined table relationships, primary and foreign keys, and strong separation of concerns to ensure scalability and maintainability. In addition to the core schema, I implemented views, stored procedures, functions, triggers, transactions, and scheduled events to automate routine processes, enforce business rules, and maintain data consistency across the system. This work aligned closely with the course's focus on architecture, testing strategies, security, and reliable system design.

Through this project, I strengthened my skills in SQL development, backend system architecture, database integrity, and software engineering best practices, while gaining hands-on experience applying lifecycle concepts such as structured design, version control workflows, and maintainable system development.
